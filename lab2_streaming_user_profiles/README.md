# Анализ профилей пользователей стримингового сервиса

Проект машинного обучения для анализа поведения пользователей онлайн-кинотеатра / стримингового сервиса.

## Возможности проекта

- кластеризация пользователей;
- PCA-визуализация;
- t-SNE-визуализация;
- DBSCAN-анализ;
- Agglomerative Clustering;
- прогноз оттока пользователя;
- классификация типа пользователя;
- построение графиков;
- сохранение моделей;
- итоговый отчет.

## Описание датасета

Каждая строка — профиль пользователя стримингового сервиса.

Признаки:

- возраст;
- часы просмотра в месяц;
- средняя длительность сессии;
- количество сессий;
- фильмы / сериалы / документальные фильмы;
- длительность подписки;
- стоимость подписки;
- обращения в поддержку;
- дни с последнего входа;
- лайки;
- оценки;
- элементы в списке просмотра;
- количество устройств.

Целевые переменные:

- churn — отток пользователя;
- user_type — тип пользователя.
